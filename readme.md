## Script i use as esx_TextUI




# Example Code

<h3>Change style type</h3>

```lua
---usage: message/type
ESX.TextUI("i ~r~love~s~ donuts", "error")
ESX.TextUI("i ~g~love~s~ donuts", "success")
ESX.TextUI("i ~b~love~s~ donuts", "info")
```

<h3>Hide the TextUI</h3>

```lua
ESX.HideUI()
```

<h3>Color Code Usage</h3>

```lua
~r~ = Red
~b~ = Blue
~g~ = Green
~y~ = Yellow
~p~ = Purple
~c~ = Grey
~m~ = Dark Grey
~u~ = Black
~o~ = Orange

ESX.TextUI("i ~r~love~s~ donuts", "error")
```
